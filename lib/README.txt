# 依赖JAR包目录

本目录需放置以下JAR包：

1. **mysql-connector-j-8.x.jar** - MySQL JDBC驱动
   - 下载地址：https://dev.mysql.com/downloads/connector/j/

2. **poi-5.x.jar** - Apache POI核心库（Excel读取）
   - 下载地址：https://poi.apache.org/download.html

3. **poi-ooxml-5.x.jar** - Apache POI OOXML支持
   - 下载地址：https://poi.apache.org/download.html

4. **poi-ooxml-lite-5.x.jar** - Apache POI OOXML Lite

以及相关的xmlbeans、commons-collections4、commons-compress、commons-math3等依赖。

建议使用Maven管理依赖，或从Apache POI官网下载完整包。
